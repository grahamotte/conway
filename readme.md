# Running

ruby:
```
  ruby conway.rb
```

elixir:
```
  elixir conway.ex
```

crystal:
```
  crystal run conway.cr
```

node:
```
  node conway.js
```

python:
```
  python3 conway.py
```

bash:
```
  bash conway.sh
```

c:
```
  gcc -o conway conway.c && ./conway
```

go:
```
  go run conway.go
```
